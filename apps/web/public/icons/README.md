# PWA icons

Required asset files (referenced by `public/manifest.webmanifest`):

- `icon-192.png` — 192×192, square, full-bleed Tierra mark (any-purpose).
- `icon-512.png` — 512×512, square, full-bleed Tierra mark (any-purpose).
- `icon-maskable-192.png` — 192×192, maskable icon with ≥10% safe zone padding.
- `icon-maskable-512.png` — 512×512, maskable icon with ≥10% safe zone padding.

Design source: `docs/brand/09-logo.md` (in progress). Until the final logo is locked, drop in a temporary monogram following `docs/brand/02-color.md` palette (background `#FFFCF7`, mark `#A85A2A`).

The PWA install card and Lighthouse PWA audit will fail until these files are present.
