# 01 — Worker Onboarding: i18n

All strings live in `packages/i18n/{en,es}.json` under the `worker.onboarding.*` namespace.

> **Inferred:** Spanish translations target Mexican-Spanish farmworker idiom rather than Castilian or formal LatAm Spanish, since the user base is overwhelmingly from rural Mexico. A native-speaker review pass is a hard ship blocker — these strings are starting drafts that should be reviewed by at least one community partner before launch.
