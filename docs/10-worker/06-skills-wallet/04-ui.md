# 06 — Skills Wallet: UI

## Routes

| route                                  | screen                           |
| -------------------------------------- | -------------------------------- |
| `/[locale]/wallet`                     | wallet list                      |
| `/[locale]/wallet/cert/[enrollmentId]` | individual cert preview/download |
| `/[locale]/verify/[certificateId]`     | public verify page (Phase 2)     |

## Wallet list

```
│  │   Fresno County WIB        ││
│  │   [Download] [Share]       ││

│  ┌────────────────────────────┐│
│  │ ★ Pesticide Applicator     ││
│  │   Self-reported            ││
│  │   Issued Mar 2024          ││
│  └────────────────────────────┘│
└────────────────────────────────┘
```

- Each card shows: title (active locale), source badge ("AgConn" with star vs. "Self-reported"), date, org, action buttons.
- Tap a card → `/wallet/cert/<id>` with full preview.
- Self-reported entries (manual) don't have Download / Share since no PDF exists.

```
┌────────────────────────────────┐
│  ← Back to wallet              │
│  Tractor Safety                │
│  Capacitación de Seguridad     │
│  en Tractor                    │
│                                │
│  Cert ID: AC-2026-X7K2P9       │
│  Issued: Apr 12, 2026          │
│  By: Fresno County WIB         │
│  Funder: CDFA                  │
│                                │
│  [Embedded PDF preview]        │
│                                │
│  [Download] [Share]            │
└────────────────────────────────┘
```

PDF preview: use `<object>` or `<embed>` tag with the signed Blob URL, with fallback "Open PDF" link for browsers that don't render embedded PDFs (mobile Safari hides the embed).

## Share menu

System share API where available (`navigator.share` on mobile); fallback dropdown for desktop:

- WhatsApp: `https://wa.me/?text=<encoded share text>`
- SMS: `sms:?&body=<encoded share text>`
- Copy link: `navigator.clipboard.writeText(shareUrl)`

## States

- **Empty wallet:** "You haven't earned any certifications yet. [Browse training]"
- **Loading:** skeleton cards.
- **Cert deleted (rare):** "This certificate is no longer available. Contact support."

## Components

- `<WalletList>`, `<WalletCard>`, `<CertPreview>`, `<ShareMenu>` — in `apps/web/app/[locale]/wallet/_components/`.

## Performance

- Wallet list RSC; signed URLs generated server-side (24h validity).
- Cert preview lazy-loads the PDF — placeholder until iframe/embed loads.

## Accessibility

- PDF preview has descriptive heading ("Certificate of Completion: Tractor Safety").
- Download button has explicit `<a download="...">` for screen readers.
- Share menu items have proper button roles.
