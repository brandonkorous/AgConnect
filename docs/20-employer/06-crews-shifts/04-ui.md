# 06 · Crews & Shifts — UI

Page: `/employer/crews`. Layout matches [docs/design/project/employer-pages.jsx#CrewsPage](../../design/project/employer-pages.jsx).

## Sections (top → bottom)

1. **Header** — eyebrow ("Operations · Week of {date}"), title "Crews & shifts", summary line (n crews · n confirmed · n still filling · n hours scheduled), right-side actions: Export schedule, New shift.
2. **5-day schedule grid** — sticky header row (Mon–Fri date), each row is a crew, each cell is a shift slot. Tone tokens: `work` (primary‑soft), `pending` (warning), `note` (base‑warm). Empty cell = "Off" (dashed border, muted).
3. **Crew leaders strip** — 4 cards (foreman initials, crew name, size, rating, "Message {firstName}" button).

## Components

- `CrewsScheduleGrid({ crews, days, shiftsByCrewByDay, locale })` — pure render
- `CrewLeaderCard({ crew, foreman })`
- `NewShiftModal` (`'use client'`) — POST `/v1/employer/shifts`
- `AssignWorkerModal` — pulls hired-applicant list

## Create vs. edit recurrence

The **create page** uses `SeriesDateSection` — a start date, end date, and a
7-toggle weekday picker. A live "Creates N shifts" preview reflects the resolved
date count. On save: a single resolved date posts to `POST /v1/employer/shifts`
(plain one-off, `seriesId` null); more than one posts to
`POST /v1/employer/shifts/series`.

The **single-shift edit page** has no recurrence UI at all — `DateTimeSection` is
date + status + start/end time only. Editing a shift never spawns siblings.
Series-level management (bulk edit, range cancel/extend) is a separate Phase 2
surface.

## Daisy / brand mapping

- "Crew A · Grape harvest" pill in header → `bg-primary/10 text-primary`
- Schedule cell tones use Tierra theme tokens (no hardcoded hex). `work` = `bg-primary/15 text-primary border-l-primary`; `pending` = `bg-warning/15 text-warning-content border-l-warning`.
- Use `font-display` for crew title in cell, `font-mono` for size badge.

## Empty states

- No crews: full-card invitation "Create your first crew" → opens `NewCrewModal`
- No shifts in window: "Schedule shifts to see your operations week"
