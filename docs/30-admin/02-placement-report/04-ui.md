# 02 — Placement Report: UI

Admin app. English-only.

## Routes

| route | screen |
|---|---|
| `/placements` | report builder + preview |
| `/reports/runs` | historical exports list |

## Report builder

```
┌─────────────────────────────────────┐
│  Placement Report                   │
│                                     │
│  Filters                            │
│  Tenant: [All ▾]                    │
│  County: [All ▾] [Fresno ✓] [Kern ✓]│
│  Funder: [Any ▾]                    │
│  Date range: [Start] – [End]        │
│  ☐ Include worker names             │
│                                     │
│  Format: ( ) CSV  ( ) XLSX           │
│  Delivery: ( ) Download             │
│             ( ) Email me            │
│                                     │
│  [Preview]  [Generate]              │
│                                     │
│  Preview (first 20 rows of 142)     │
│  ┌─────────────────────────────────┐│
│  │ scrollable table of preview     ││
│  └─────────────────────────────────┘│
└─────────────────────────────────────┘
```

Preview updates live as filters change (debounced 500ms).

Generate → either downloads file or shows "Sent to your email" toast.

## Historical runs

```
┌─────────────────────────────────────┐
│  Past reports                       │
│  [Filter: type ▾] [Date range]      │
│                                     │
│  ┌─────────────────────────────────┐│
│  │ Placement (CSV) — 142 rows      ││
│  │ Apr 12, 2026 by admin@...       ││
│  │ Q1 2026, all counties           ││
│  │ [Re-download]                   ││
│  └─────────────────────────────────┘│
│  ...                                │
└─────────────────────────────────────┘
```

## States

- **Empty preview:** "No placements match these filters."
- **Date range too wide:** form blocks generation; tooltip on the Generate button.
- **Generation in progress:** spinner with "Building your report..." message.
- **Email delivery:** "Your report is on its way. Check your inbox." with link to runs page.

## Performance

- Preview RSC; query takes < 1s typically.
- Full export server-side; sub-30s for typical sizes.
- Client receives binary file via standard download (browser native).

## Components

- `<PlacementReportBuilder>`, `<ReportPreviewTable>`, `<RunsList>`, `<DateRangePicker>`, `<FilterChips>` — in `apps/admin/app/placements/_components/`.

## Accessibility

- Preview table: `<table>` with `<thead>` and proper labels.
- All filter controls keyboard-accessible.
- "Generate" disabled state has clear `aria-disabled` + tooltip.
