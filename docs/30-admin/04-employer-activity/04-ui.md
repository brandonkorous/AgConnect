# 04 — Employer Activity Report: UI

Admin app. English-only.

## Routes

| route | screen |
|---|---|
| `/employers` | report builder + preview |
| `/employers/:id` | per-employer drill-down |
| `/reports/runs` | shared with placement-report |

## Report builder

Mirrors the placement / training reports. Filters: tenant, county, license type, date range, view selector (rows / by_county / by_license_type), format, delivery.

```
┌─────────────────────────────────────┐
│  Employer Activity Report           │
│                                     │
│  View: ( ) Per employer             │
│        ( ) By county                │
│        ( ) By license type          │
│                                     │
│  Filters [...]                      │
│  [Preview] [Generate]               │
│                                     │
│  Preview                            │
│  ┌───────────────────────────────┐  │
│  │ Central Valley Farms FLC      │  │
│  │ Fresno • 12 hires • 22% rate  │  │
│  │ Avg wage $19.40 / 4 days     │  │
│  └───────────────────────────────┘  │
│  ...                                │
└─────────────────────────────────────┘
```

- Empty preview, loading, generation progress, email confirmation — same patterns.
