# 06 — County Landing Pages: i18n

EN and ES ship together, full parity, both authored (ES is **not** a templated
translation of EN — both languages are native). Copy lives in the **landing**
seed bundle (`packages/db/seed-data/translations/landing.ts`), DB-backed:
edit the bundle + reseed, never the generated `messages/{en,es}.json`
([[feedback-bilingual-by-design]]).

## Key namespace

Per county slug `c` ∈ {fresno, tulare, kern, kings, madera}:

| key | purpose | notes |
| --- | --- | --- |
| `county.{c}.h1` | page H1 | upright, no word-art |
| `county.{c}.meta_title` | `<title>` / OG title | ≤ ~60 chars, includes county + "farm jobs" |
| `county.{c}.meta_description` | meta description | ≤ ~155 chars, answer-style |
| `county.{c}.intro_template` | answer paragraph | placeholders `{count}` `{median}` `{topRole}` |
| `county.{c}.intro_template_no_median` | < 3 jobs variant | no `{median}` placeholder |
| `county.{c}.intro_empty` | 0 jobs variant | calm, no placeholders |
| `county.{c}.context_crops` | top crops sentence | static editorial |
| `county.{c}.context_seasons` | typical seasons sentence | static editorial |
| `county.{c}.faq.q1..q3` / `.a1..a3` | county FAQ | feeds `FAQPage` JSON-LD |
| `county.nav.all` | "All counties" cross-link | shared, not per-county |
| `county.nav.siblings_label` | "Other counties" heading | shared |
| `county.breadcrumb.jobs` | breadcrumb "Jobs" segment | reuse existing jobs label if present |

## Placeholder contract

`intro_template` is interpolated at render with already-localized values:

- `{count}` — integer, locale-formatted (Inter tabular-nums in UI).
- `{median}` — currency string formatted server-side per locale (`$19.50/hr` /
  `$19.50/h`). The string is passed in pre-formatted; the template does not
  format.
- `{topRole}` — the most frequent job title; **Inferred:** title strings come
  from job data (employer-entered) and are shown as-is, not translated, since
  job titles are user-generated content (per CLAUDE.md UGC exception). The
  surrounding sentence is localized; the role noun is not.

EN example: `There are {count} seasonal farm jobs in Fresno County right now,
with a median of {median}. Most are {topRole} roles.`

ES example (authored, not a literal mirror): `Hay {count} trabajos agrícolas de
temporada en el Condado de Fresno ahora mismo, con una mediana de {median}. La
mayoría son puestos de {topRole}.`

## Parity enforcement

The existing bilingual parity allowlist check ([[feedback-bilingual-by-design]])
must include every `county.*` key — a missing ES county string blocks the
build, same as any other landing string. No county page may render with an
EN-only intro.

## Register

Plain-spoken, specific, civic — county name + real numbers, no marketing verbs
in H1/meta. "Seasonal farm jobs in Kern County", not "Unlock Kern County
opportunities". Eyebrow labels may carry a benefit verb only if paired with a
concrete line, per the brand voice rules.
