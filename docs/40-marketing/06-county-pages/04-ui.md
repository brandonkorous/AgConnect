# 06 — County Landing Pages: UI

Route: `apps/web/src/app/[locale]/jobs/county/[county]/page.tsx` (RSC, ISR).
Brand system: Tierra, civic-utilitarian. Reuses existing components; no new
brand-critical sections are hand-built for this feature.

## Page structure (top → bottom)

1. **Breadcrumb** — `Jobs / Fresno County` (`Trabajos / Condado de Fresno`).
   Renders `BreadcrumbList` JSON-LD (see [09-seo-aio](../../00-foundation/09-seo-aio/)).
2. **County header band** (`base-100` on `base-300`, hairline rule — the
   default product-card lift, no ad-hoc elevation):
   - `H1`: `Seasonal farm jobs in Fresno County` / `Trabajo agrícola de
     temporada en el Condado de Fresno`. Inter, upright (no italic display per
     [[feedback-tierra-not-editorial]]).
   - **Answer paragraph** directly under H1: the filled `intro_template` —
     `There are 48 seasonal farm jobs in Fresno County right now, with a median
     of $19.50/hr. Most are strawberry harvest, tractor operator, and irrigation
     roles.` Numbers use Inter `tabular-nums slashed-zero`. This paragraph is
     the AIO answer target — it must be real text in the DOM, server-rendered,
     not behind a tab or accordion ([[feedback-paper-fidelity]] — no opacity on
     text, no chrome hiding content).
   - Stat row: job count + median wage as two quiet stats (no confetti, color
     with intent only).
3. **Job list** — the existing `/jobs` listing component with the county filter
   fixed and the filter UI hidden (county is the page identity, not a toggle).
   Pill filter chips for crop/sort may remain if the component already exposes
   them; do **not** build a parallel list.
4. **County context block** — short bilingual editorial: top crops, typical
   seasons, common roles. `base-200` band. Plain-spoken register (UC Davis
   extension, not SaaS).
5. **County FAQ** — 3 Q&A from the i18n bundle, rendered as visible text
   (daisyUI not required to be an accordion; if collapsed, content stays in DOM
   for crawlers). Emits `FAQPage` JSON-LD.
6. **Sibling-county cross-links** — pill links to the other four counties +
   "All counties" → `/jobs`. Internal link equity; mirrors the footer county
   strip which now points at these same routes.

## Empty / low-data states

- **0 active jobs in county**: keep the page (it must stay indexable and not
  404 — see [08-edge-cases](08-edge-cases.md)). Replace the list with a calm
  bilingual line — `No open postings in Fresno County today. New jobs are
  posted through the season — check back, or see all counties.` — plus the
  cross-links and county-context block. No urgency theater.
- **< 3 jobs**: render count, suppress median (data-model rule); intro template
  uses the no-median variant string.

## Accessibility / brand invariants

- 44×44 touch targets on every county/sort pill; visible `:focus-visible`.
- County name never communicated by color alone.
- WCAG 2.1 AA min; body AAA where palette permits.
- No emoji. FontAwesome per-icon only. Canonical container
  `container mx-auto px-5 md:px-8 lg:px-20`; reading caps on children only.
- `prefers-reduced-motion` respected (there is no motion here by default —
  static layout that breathes).
